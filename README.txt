 【 TubeInvestigation AI 】 セットアップガイド

この度は、YouTubeリサーチ特化型AIツール「TubeInvestigation AI」をご購入いただき、誠にありがとうございます。
本ファイルでは、ツールの起動方法とAPIキーの取得・設定についてご説明します。

ーーーーーーーーーーーーーーー
■ 1. 初回起動の手順
1.ファイルの展開: ダウンロードしたZipファイルをすべて展開（解凍）してください。
2.実行ファイルの起動: 
  ・Windowsの場合: フォルダ内の TubeInvestigationAI.exe をダブルクリックします。
  ・Macの場合: フォルダ内の TubeInvestigationAI.app をダブルクリックして起動します。
3.設定画面の入力: 起動後に表示されるウィンドウに、以下の情報を入力してください。
・ライセンスキー: メールで発行されたキーをコピー＆ペーストしてください。
・YouTube Data API v3 キー: 動画データの取得に使用します。
・Gemini APIキー: AI分析に使用します。

4.開始: 「設定を保存して開始」ボタンを押し、認証が通れば利用可能になります。

ーーーーーーーーーーーーーーー
■ 2. 各APIキーの取得について
本ツールの動作には2種類のGoogle APIキーが必要です。以下のいずれかの方法でご準備ください。

・事前に取得する場合: Google AI Studio（Gemini API）および Google Cloud Console（YouTube Data API v3）より取得してください。
・起動後に取得する場合: ツール起動時のライセンス入力ウィンドウ内に、各APIキー取得ページへの直通リンクを設置しております。取得いただけます。

ーーーーーーーーーーーーーーー
■ 3. APIの利用制限について（重要）
・YouTube Data API v3: 通常の検索には十分な余裕がありますが、1日に数百回も利用すると利用制限になる可能性がございます。
・Gemini API (無料枠): 1日あたり10回程度の分析（計20リクエスト）が目安です。
※ より多くの分析が必要な場合は、Google AI Studioにて有料プランへのアップグレードをご検討ください。

ーーーーーーーーーーーーーーー
